Codeforces Round 896 (Div. 2) C题题解

C. Fill in the Matrix

C. Fill in the Matrix 题解

特判 $m=1$ 时，输出 $0$

找规律，当 $n$ 很大时， $ans=m$ ，$m$ 很大的时候， $ans=n+1$

即最后的矩阵第一列中出现 $0$ 不出现 $1$ ，第二列不出现 $0$ ，第三列出现 $0$ ， $1$ 不出现 $2$ ，最终的 $v_1=1,v_2=0,v_3=2···v_m=m-1$

即构造出如下矩阵：

0 1 2 3 4
4 0 1 2 3
3 4 0 1 2
2 3 4 0 1
0 1 2 3 4
4 0 1 2 3
3 4 0 1 2

注：第一列不出现 1
